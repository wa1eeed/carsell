# دورة حياة السيارة — Car Lifecycle

---

## حالات السيارة

```
┌──────────┐    حفظ النموذج    ┌──────────┐
│  مسودة   │ ────────────────► │  للبيع   │
│  draft   │                   │ for_sale  │
└──────────┘                   └────┬─────┘
                                    │
                     ┌──────────────┼──────────────┐
                     ▼              ▼               ▼
              ┌────────────┐  ┌──────────┐  ┌───────────┐
              │ على السوم  │  │ محجوزة   │  │   مباعة   │
              │   auction  │  │ reserved │  │   sold    │
              └────────────┘  └──────────┘  └───────────┘
                     │              │
                     └──────────────┘
                           │
                           ▼
                     ┌──────────┐
                     │   مباعة  │
                     │   sold   │
                     └──────────┘
```

---

## تفاصيل كل حالة

| الحالة | اللون | المعنى | الإجراءات المتاحة |
|---|---|---|---|
| `draft` | رمادي | مُدخلة لكن غير معروضة | تعديل، نشر، حذف |
| `for_sale` | أزرق | معروضة للبيع بسعر ثابت | تعديل، حجز، بيع، سحب |
| `auction` | بنفسجي | معروضة للسوم | تعديل، قبول عرض، سحب |
| `reserved` | برتقالي | محجوزة لعميل | تأكيد بيع، إلغاء الحجز |
| `sold` | أخضر | تمّ البيع | عرض فقط — لا تعديل |

## التايم لاين (Audit Log)

كل حدث على السيارة يُسجَّل تلقائياً:

### أنواع الأحداث

| نوع الحدث | الأيقونة | مثال |
|---|---|---|
| `car_created` | ➕ | إدخال السيارة للمخزون |
| `field_updated` | ✏️ | تعديل سعر البيع من X إلى Y |
| `status_changed` | 👁 | تغيير الحالة: مسودة → للبيع |
| `file_uploaded` | 📎 | رفع ملف: تقرير_الفحص.pdf |
| `file_deleted` | 🗑 | حذف ملف: صورة_٣.jpg |
| `sale_registered` | 🧾 | تسجيل بيع — العميل: محمد الأحمدي |
| `price_changed` | 💰 | تغيير السعر من 300,000 إلى 320,000 |
| `note_added` | 💬 | إضافة ملاحظة |

### هيكل سجل التايم لاين

```typescript
interface TimelineEvent {
  id: string
  car_id: string
  event_type: EventType
  user_id: string
  user_name: string
  timestamp: DateTime
  payload: {
    field?: string          // الحقل المتغيّر
    old_value?: any         // القيمة القديمة
    new_value?: any         // القيمة الجديدة
    description?: string    // وصف حر
    file_name?: string      // اسم الملف إن وُجد
  }
}
```

## المستندات المرتبطة بالسيارة

| نوع المستند | الوصف | مثال اسم الملف |
|---|---|---|
| `inspection` | تقرير الفحص الشامل | فحص_٢٠٢٦.pdf |
| `purchase_invoice` | فاتورة الشراء | فاتورة_شراء.pdf |
| `insurance` | وثيقة التأمين | تأمين_٢٠٢٦.pdf |
| `registration` | رستورة / استمارة | استمارة.pdf |
| `previous_invoice` | فواتير صيانة سابقة | صيانة_٢٠٢٥.pdf |
| `other` | أي مستند آخر | — |

**قواعد الملفات:**
- الصيغ المقبولة: PDF, JPG, PNG, HEIC
- الحد الأقصى: 10MB للملف الواحد
- التخزين: S3-compatible storage مع توقيع مؤقت للروابط

## صفحة التفاصيل — التابز

1. **التفاصيل** — كل بيانات السيارة + الصور

3. **المستندات** — كل الملفات المرفوعة
4. **المالية** — حساب التكاليف والضريبة والربح
