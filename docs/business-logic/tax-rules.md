# قواعد الضريبة — VAT Rules

> المرجع: هيئة الزكاة والضريبة والجمارك (ZATCA)
> سارية من: ١ يوليو ٢٠٢٣

---

## الحالتان المدعومتان

### ١. الضريبة على كامل سعر البيع (الطريقة الاعتيادية)

**تطبّق على:**
- جميع السيارات الجديدة
- السيارات المستعملة غير المؤهلة
- أي معرض لم يحصل على موافقة ZATCA لطريقة الهامش

**الحساب:**
```
VAT = سعر_البيع × 15%
صافي_الربح = سعر_البيع − إجمالي_التكلفة − VAT
```

**مثال:**
```
سعر الشراء:    100,000 ر.س
تكاليف إضافية:   2,000 ر.س
إجمالي التكلفة: 102,000 ر.س
سعر البيع:     125,000 ر.س
VAT (15%):      18,750 ر.س
صافي الربح:      4,250 ر.س
```

---

### ٢. الضريبة على هامش الربح (Profit Margin Method)

**تطبّق على:**
- السيارات المستعملة المؤهلة (used_qualified) فقط

**شروط التأهّل (ZATCA):**
1. السيارة مصنّفة "مستعملة مؤهلة" من قِبل الهيئة
2. سبق استعمالها داخل المملكة
3. المعرض مسجّل لدى هيئة الزكاة لأغراض VAT
4. المعرض مرخّص بمزاولة نشاط تجارة السيارات
5. المعرض حاصل على **موافقة مسبقة** من الهيئة لاستخدام هذه الطريقة

**الحساب:**
```
هامش_الربح = سعر_البيع − سعر_الشراء  ← (ليس إجمالي التكلفة)
VAT = هامش_الربح × 15%
صافي_الربح = سعر_البيع − إجمالي_التكلفة − VAT
```

**مثال:**
```
سعر الشراء:    100,000 ر.س
تكاليف إضافية:   2,000 ر.س
إجمالي التكلفة: 102,000 ر.س
سعر البيع:     125,000 ر.س
هامش الربح:     25,000 ر.س  ← (125k − 100k)
VAT (15%):       3,750 ر.س  ← (25k × 15%)
صافي الربح:     19,250 ر.س
```

> ملاحظة: الفرق الضريبي بين الطريقتين في هذا المثال = 15,000 ر.س لصالح المعرض.

## إعداد المعرض في المنصة

كل معرض له إعداد ضريبي خاص به في `showroom_settings`:

```
vat_method: "full_price" | "profit_margin"
zatca_registered: boolean
profit_margin_approved: boolean
vat_number: string
```

منطق المنصة عند حساب الضريبة لأي سيارة:

```
if (car.type === "new") {
  → طريقة سعر كامل دائماً
}

if (car.type === "used_qualified"
    && showroom.profit_margin_approved === true) {
  → طريقة هامش الربح
} else {
  → طريقة سعر كامل
}
```

---

## حقل `car_type` في نموذج الإدخال

| القيمة | العرض في الواجهة | طريقة الضريبة |
|---|---|---|
| `new` | جديدة | سعر كامل |
| `used` | مستعملة | سعر كامل |
| `used_qualified` | مستعملة مؤهلة | هامش ربح (إن كان الإذن متوفراً) |

## تقارير VAT

المنصة تولّد تقارير دورية (شهرية/ربع سنوية) تتضمن:
- إجمالي VAT المحصّل
- تفكيك الطريقتين (هامش / كامل)
- قابلة للتصدير بصيغة Excel/PDF للرفع على بوابة ZATCA
