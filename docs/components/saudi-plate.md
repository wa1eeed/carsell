# مكوّن لوحة السيارة السعودية — SaudiPlate

> مقتبس ومطوَّر من مشروع نقله (wa1eeed)
> آخر تحديث: يونيو 2026

---

## الوصف

مكوّن React يعرض لوحة السيارة السعودية بشكلها الحقيقي، مع دعم الإدخال التفاعلي ومعاينة فورية.

---

## Layout اللوحة الحقيقية

```
┌─────────────────────────────────────────┐
│  ١١١١  │  🌴⚔️  │  ح ن ط              │
│        │  KSA   │                       │
│  1111  │السعودية│  T N J               │
└─────────────────────────────────────────┘
  أرقام    شريط      حروف
  (يسار)  (وسط)    (يمين RTL)
```

**ملاحظات التصميم:**
- ٣ أعمدة: أرقام | شريط KSA | حروف
- كل عمود منقسم صفّين: عربي (فوق) + لاتيني (تحت) بخط فاصل
- الشريط الوسطي: شعار المملكة + KSA + السعودية + مثلث ▼
- لون الشريط يحدد نوع اللوحة

## أنواع اللوحات

| النوع | اللون | القيمة |
|---|---|---|
| خاصة | أخضر `#2E7D32` | `private` |
| أجرة / نقل عام | أصفر `#F5C518` | `taxi` |
| نقل | كحلي `#1B3A6B` | `transport` |
| دبلوماسية | أزرق `#1565C0` | `diplomat` |

## الحروف المعتمدة — ١٧ حرفاً رسمياً

مصدر: هيئة المرور السعودية

| عربي | لاتيني | عربي | لاتيني | عربي | لاتيني |
|---|---|---|---|---|---|
| أ | A | ص | X | م | Z |
| ب | B | ط | T | ن | N |
| ح | J | ع | E | هـ | H |
| د | D | ق | G | و | U |
| ر | R | ك | K | ي | V |
| س | S | ل | L | | |

**الحروف المستبعدة:**
ت، ث، ج، ذ، ز، ش، ض، ظ، غ، ف، خ، ة، ء، ى

## Props

```typescript
interface SaudiPlateProps {
  letters?: string[]          // حتى 3 حروف عربية من القائمة المعتمدة
  numbers?: string            // 1-4 أرقام غربية (0-9)
  type?: PlateType            // private | taxi | transport | diplomat
  size?: 'sm' | 'md' | 'lg'
  interactive?: boolean       // يُظهر حقول الإدخال
}

type PlateType = 'private' | 'taxi' | 'transport' | 'diplomat'
```

---

## القيمة المُخزَّنة في قاعدة البيانات

```typescript
// ما يُحفظ
plateNumber: "ح ن ط 7653"   // حروف بمسافات + أرقام غربية
plateType: "private"         // نوع اللوحة — لا يُفقد خلافاً لمشروع نقله

// ما يُستنتج عند العرض
lettersEn: ["J", "N", "T"]  // مستنتج من جدول SAUDI_PLATE_LETTERS
lettersEnReversed: ["T", "N", "J"]  // للعرض (RTL → LTR)
```

**⚠️ درس من مشروع نقله:** `plateType` كان يُفقد لعدم إضافته للـ API payload.
في هذه المنصة يُحفظ صراحةً في جدول `cars`.

## ملاحظات للتطوير

- الحروف اللاتينية **تُعكس** عند العرض (RTL → LTR): `["ح","ن","ط"]` → عربي: `ح ن ط` → لاتيني: `J N T` لكن يُعرض `T N J`
- الأرقام تُخزَّن غربية دائماً، وتُحوَّل للعربية-الهندية عند العرض فقط via `toArabicDigits()`
- المكوّن يجب أن يكون في `packages/shared-ui` (حزمة مشتركة) — لا يُكرَّر per-portal
