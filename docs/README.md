# CarLink — سوق السيارات — توثيق المشروع

> **الإصدار:** 0.1 — مرحلة التصميم والتوثيق
> **آخر تحديث:** يونيو 2026

---

## نظرة عامة

منصة SaaS متعددة المستأجرين (multi-tenant) مصممة خصيصاً لمعارض السيارات في السوق السعودي. تتيح لأصحاب المعارض إدارة المخزون، تتبع المبيعات، وحساب الأرباح والضرائب بما يتوافق مع أنظمة هيئة الزكاة والضريبة والجمارك (ZATCA).

## المستخدمون

| المستخدم | الصلاحيات |
|---|---|
| **Platform Admin** | إدارة كل المعارض، إعداد البراندات والفئات، إعدادات الضريبة |
| **معرض (Owner/Manager)** | إدارة مخزونه كاملاً، المبيعات، التقارير |
| **موظف المعرض** | إدخال سيارات، تحديث الحالات، رفع مستندات |

## الحالات الرئيسية للسيارة

```

              ↘ على السوم ↗
```
