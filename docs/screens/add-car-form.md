# نموذج إدخال سيارة جديدة — Add Car Form

---

## الوصف

نموذج متعدد الأقسام لإضافة سيارة للمخزون. يعمل بنظام الـ cascading dropdowns المرتبطة بإعدادات الأدمن، مع حساب الأرباح والضريبة في الوقت الفعلي.

---

## أقسام النموذج

### القسم ١: تحديد السيارة

| الحقل | النوع | المصدر | إلزامي |
|---|---|---|---|
| البراند | dropdown | `brands` table (admin) | ✓ |
| الفئة | dropdown (cascading) | `categories` by brand | ✓ |
| الموديل | dropdown (cascading) | `models` by category | ✓ |
| سنة الصنع | dropdown | 2000 → سنة الحالية | ✓ |
| نوع السيارة | dropdown | new / used / used_qualified | ✓ |
| رقم الهيكل VIN | text | حر | — |

**منطق الـ Cascading:**
```
اختيار براند → تفعيل قائمة الفئات (تُصفَّى حسب brand_id)
اختيار فئة  → تفعيل قائمة الموديلات (تُصفَّى حسب category_id)
              + إظهار نوع الهيكل (SUV / سيدان / بيك آب) تلقائياً
```

### القسم ٢: المواصفات

| الحقل | النوع | إلزامي |
|---|---|---|
| اللون الخارجي | dropdown (من إعدادات الأدمن) | — |
| اللون الداخلي | dropdown | — |
| نوع الوقود | dropdown: بنزين / ديزل / هجين / كهربائي | — |
| ناقل الحركة | dropdown: أوتوماتيك / يدوي | — |
| عداد المسافات | number (كم) | — |
| هيكل السيارة | auto (من الفئة) أو dropdown | — |
| ملاحظات | textarea | — |

### القسم ٣: التسعير

| الحقل | النوع | إلزامي |
|---|---|---|
| سعر الشراء | number | ✓ |
| تكاليف إضافية | number (قابل للتفصيل) | — |
| طريقة العرض | toggle: سعر ثابت / على السوم | ✓ |
| سعر البيع | number (يُخفى عند السوم) | إن ثابت |
| حالة العرض | select: للبيع / سوم / محجوزة / مسودة | ✓ |

### القسم ٤: لوحة السيارة

| الحقل | القاعدة |
|---|---|
| نوع اللوحة | private / taxi / transport / diplomat |
| الحرف الأول | إلزامي — من القائمة الرسمية (١٧ حرف) |
| الحرف الثاني | اختياري |
| الحرف الثالث | اختياري |
| الأرقام | إلزامي — ١ إلى ٤ أرقام |

### القسم ٥: الصور والمستندات

- رفع حتى ٢٠ صورة (JPG/PNG، حد أقصى ٥MB للصورة)
- مستندات: فحص، فاتورة شراء، تأمين، رستورة، أخرى
- كل ملف يُسجَّل في التايم لاين عند رفعه

## الملخص الجانبي (Real-time Sidebar)

يتحدث مع كل تغيير في حقول التسعير:

```
سعر الشراء:         290,000
+ تكاليف إضافية:      4,500
= إجمالي التكلفة:   294,500
  سعر البيع:        345,000
- VAT (حسب النوع):    8,250   ← يتغير تلقائياً
= صافي الربح:       +42,250  ← أخضر إن ربح، أحمر إن خسارة
  نسبة الربح:        14.4%
```

## Validation Rules

```
البراند:       required
الفئة:         required
الموديل:       required
سنة الصنع:    required
نوع السيارة:  required
سعر الشراء:   required, min: 1
سعر البيع:    required if طريقة العرض = ثابت
لوحة — الحرف الأول: required, من SAUDI_PLATE_LETTERS
لوحة — الأرقام:     required, 1-4 digits
```

---

## التايم لاين عند الحفظ

عند حفظ السيارة لأول مرة تُسجَّل الحدث التالية:
```
✦ إدخال السيارة للمخزون — [اسم المستخدم] — [التاريخ والوقت]
```
كل تعديل لاحق يُسجَّل بالتفصيل (ما تغيّر، من غيّر، متى).
