# صفحة تسجيل البيع — Sale Registration

## الوصف
صفحة تُفتح من زر "تسجيل بيع" في صفحة تفاصيل السيارة. تسجّل البيع، تحسب الضريبة والربح، وتولّد عقد البيع.

## بيانات المشتري
| الحقل | إلزامي |
|---|---|
| الاسم الكامل | ✓ |
| رقم الجوال | ✓ |
| رقم الهوية / الإقامة | — |
| البريد الإلكتروني | — |

## تفاصيل البيع
- سعر البيع الفعلي (قابل للتعديل، prefilled من سعر السيارة)
- تاريخ البيع
- طريقة الدفع: نقداً / تحويل / تمويل / مختلط
- ملاحظات العقد

## الحسابات التلقائية
```
VAT = (sell - purchase) * 15%        ← مستعملة مؤهلة
VAT = sell * 15%                      ← جديدة / مستعملة عادية
netProfit = sell - totalCost - VAT
customerTotal = sell + VAT
```

## ما يحدث عند تأكيد البيع
1. إنشاء سجل `Sale` في قاعدة البيانات
2. تحديث `car.status` → `SOLD`
3. تسجيل حدث `SALE_REGISTERED` في `CarTimeline`
4. توليد عقد PDF (Pro+)
5. إرسال نسخة للعميل عبر واتساب/بريد (اختياري)

## العقد PDF — الحقول
- بيانات المعرض (من إعدادات الشركة)
- بيانات المشتري
- مواصفات السيارة الكاملة
- سعر البيع + VAT + الإجمالي
- رقم الهيكل VIN ورقم اللوحة
- تاريخ البيع وتوقيع الأطراف
