# لوحة الأدمن — Admin Panel

---

## الوصف

لوحة تحكم المنصة الرئيسية. تُدار من قِبل فريق المنصة، وتتحكم في الإعدادات العامة التي تؤثر على جميع المعارض.

---

## أقسام الأدمن

### ١. إدارة البراندات والفئات

**هيكل البيانات:**
```
Brand (براند)
  └── Category (فئة) — مثال: Land Cruiser
        ├── body_type: SUV | سيدان | بيك آب | كوبيه | هاتشباك | فان
        └── Model (موديل) — مثال: VXR, GXR, Twin Turbo
```

**قواعد الـ Cascading في نموذج الإدخال:**
- اختيار براند → تُحمَّل فئاته فقط
- اختيار فئة → يُحدَّد نوع الهيكل (body_type) تلقائياً
- اختيار فئة → تُحمَّل موديلاتها فقط

**حقول البراند:**
| الحقل | النوع | ملاحظة |
|---|---|---|
| name_ar | string | اسم البراند بالعربي |
| name_en | string | اسم البراند بالإنجليزي |
| logo_url | image | اختياري |
| is_active | boolean | إخفاء من النماذج |

**حقول الفئة:**
| الحقل | النوع |
|---|---|
| brand_id | FK |
| name_ar | string |
| name_en | string |
| body_type | enum |
| is_active | boolean |

| القائمة | أمثلة |

### ٥. إعدادات CarLink Market

يتحكم الأدمن في الـ Marketplace من قسم مستقل:

**إعدادات المنصة العامة (`PlatformConfig`):**
| الحقل | الوصف | الافتراضي |
|---|---|---|
| `defaultCommissionPct` | نسبة العمولة الافتراضية على كل صفقة | 2.5% |
| `minCommissionPct` | الحد الأدنى — يمكن الإعفاء التام | 0% |
| `maxCommissionPct` | الحد الأقصى | 10% |
| `activePaymentGateway` | بوابة الدفع الفعّالة | moyasar |
| `vatPct` | نسبة الضريبة المضافة (ZATCA) | 15% |

**إعدادات كل معرض (override):**
- تفعيل/إيقاف CarLink Market لهذا المعرض
- عمولة مخصصة لهذا المعرض (تتجاوز الافتراضي)
- عرض حالة الـ Onboarding مع بوابة الدفع
- تعليق حساب المعرض في السوق

**السيارات في السوق:**
- `car.listedOnMarket = true` ← تظهر في CarLink Market
- `car.marketPrice` ← سعر العرض في السوق (قد يختلف عن سعر المعرض الداخلي)
- الأدمن يستطيع إخفاء أي سيارة من السوق

  - لا يرى التقارير المالية الكاملة

## لوحة السوبر أدمن — تحديث يونيو ٢٠٢٦

### الأقسام الرئيسية

**KPIs العلوية (٥):** معارض نشطة، إجمالي المخزون، مبيعات الشهر، إيراد المنصة، مستخدمون نشطون.

**CarLink Market:** مخزون معروض + قيمة السوق + مشاهدات + عمولات + توزيع البراندات.

**المزادات النشطة:** جدول بأعلى سوم + عداد زمني + حالة (نشط / قادم).

**البائعون:** donut توزيع حسب النوع (معرض/وكالة/فرد) + توزيع الباقات.

**المبيعات:** إجمالي الصفقات + قيمة + Market + توزيع طرق الدفع.

**KYC:** نفاذ موثّق + معلق + مرفوض + تنبيه فوري للمعلق.

**جدول المخزون الحديث:** آخر السيارات المضافة مع VIN + مصدر (Market/مزاد/فرد).

**التنبيهات:** KYC معلق >٢٤ساعة، مزاد بلا مزايد، اشتراكات منتهية، صفقات مكتملة.

**Activity Feed:** آخر النشاطات في المنصة بـ real-time.
